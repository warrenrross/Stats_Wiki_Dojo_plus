1 p, c, and u Charts

1.1 Learning Goals

  1. Choose among p, c, and u charts based on data type and sample size
  2. Build each chart in R using the qcc package
  3. Handle variable sample sizes in p and u charts
  4. Interpret out-of-control signals on attribute charts

1.2 Chart Selection

Chart Data Sample size
p Fraction defective (proportion) Variable or fixed
c Count of defects (per inspection unit) Fixed
u Defects per unit Variable

1.3 p-Chart (Fraction Defective)

\[\bar{p} = \frac{\sum D_i}{\sum n_i}, \quad UCL_p = \bar{p} + 3\sqrt{\frac{\bar{p}(1-\bar{p})}{n_i}}, \quad LCL_p = \max(0, \bar{p} - \cdots)\]

With variable n, limits vary by subgroup.

1.4 R Example: p-Chart

Code 
pacman::p_load(pacman, qcc)

data("orangejuice")  # D = defectives, size = sample size
head(orangejuice)
  sample  D size trial
1      1 12   50  TRUE
2      2 15   50  TRUE
3      3  8   50  TRUE
4      4 10   50  TRUE
5      5  4   50  TRUE
6      6  7   50  TRUE
Code 
p_chart <- qcc(data = orangejuice$D, type = "p",
               sizes = orangejuice$size,
               title = "p-Chart — Orange Juice Cans")
Code 
cat("\np-bar:", round(p_chart$center, 4), "\n")

p-bar: 0.1778
Code 
cat("Out-of-control points:", p_chart$violations$beyond.limits, "\n")
Out-of-control points: 13 15 21 22 23

1.5 R Example: c-Chart and u-Chart

Code 
# c-chart: fixed inspection unit (e.g. 25 printed circuit boards per shift)
# Simulated defect counts — each subgroup inspects exactly 1 unit

defects <- c(6, 4, 8, 5, 3, 7, 9, 2, 6, 5, 8, 4, 7, 6, 3, 5, 8, 6, 4, 7)

Code

c̄ (center): 5.65
Code 
cat("UCL:", round(c_chart$limits[2], 3), "\n")
UCL: 12.781
Code 
# u-chart: variable inspection area

               title = "u-Chart — Circuit Boards")

# p, c, and u Charts

## Learning Goals

1. Choose among p, c, and u charts based on data type and sample size
2. Build each chart in R using the qcc package
3. Handle variable sample sizes in p and u charts
4. Interpret out-of-control signals on attribute charts

## Chart Selection

| Chart | Data | Sample size |
|-------|------|------------|
| p | Fraction defective (proportion) | Variable or fixed |
| c | Count of defects (per inspection unit) | Fixed |
| u | Defects per unit | Variable |

## p-Chart (Fraction Defective)

$$\bar{p} = \frac{\sum D_i}{\sum n_i}, \quad UCL_p = \bar{p} + 3\sqrt{\frac{\bar{p}(1-\bar{p})}{n_i}}, \quad LCL_p = \max(0, \bar{p} - \cdots)$$

With variable n, limits vary by subgroup.

## R Example: p-Chart

```{r}
#| label: p-chart

pacman::p_load(pacman, qcc)

data("orangejuice")  # D = defectives, size = sample size
head(orangejuice)

p_chart <- qcc(data = orangejuice$D, type = "p",
               sizes = orangejuice$size,
               title = "p-Chart — Orange Juice Cans")

cat("\np-bar:", round(p_chart$center, 4), "\n")
cat("Out-of-control points:", p_chart$violations$beyond.limits, "\n")
```

## R Example: c-Chart and u-Chart

```{r}
#| label: c-chart

# c-chart: fixed inspection unit (e.g. 25 printed circuit boards per shift)
# Simulated defect counts — each subgroup inspects exactly 1 unit
set.seed(5)
defects <- c(6, 4, 8, 5, 3, 7, 9, 2, 6, 5, 8, 4, 7, 6, 3, 5, 8, 6, 4, 7)
c_chart <- qcc(data = defects, type = "c", title = "c-Chart — Circuit Boards")
cat("\nc̄ (center):", round(c_chart$center, 3), "\n")
cat("UCL:", round(c_chart$limits[2], 3), "\n")
```

```{r}
#| label: u-chart

# u-chart: variable inspection area
data("circuit")
u_chart <- qcc(data = circuit$x, type = "u",
               sizes = circuit$size,
               title = "u-Chart — Circuit Boards")
```
