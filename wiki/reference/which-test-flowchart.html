1.3 вҖңWhich t?вҖқ вҖ” The Hardest Decision

Clue in Problem Use
вҖңbefore and after on the same unitвҖқ Paired t
вҖңsame subjects tested twiceвҖқ Paired t
вҖңtwo independent groups / samplesвҖқ 2-sample t (pooled or Welch)
вҖңassume equal variancesвҖқ Pooled t
вҖңdo not assume equal variancesвҖқ Welch t
вҖңassume Пғ is knownвҖқ Z (not t)
large samples (nвӮҒ, nвӮӮ вүҘ 40) Z acceptable

1.4 Rejection Criteria at a Glance

Test Direction HвӮҒ Reject HвӮҖ if (fixed-level) P-value
Two-sided вү  |stat| > critical 2В·P(right tail)
Upper (right) > stat > critical P(right tail)
Lower (left) < stat < вҲ’critical P(left tail)

1.5 Common Trick Questions

  • вҖңIs there a difference?вҖқ вҶ’ Two-sided (вү )
  • вҖңDoes A exceed B?вҖқ вҶ’ Upper-tailed (>)
  • вҖңIs A less than B?вҖқ вҶ’ Lower-tailed (<)
  • вҖңDoes the process meet spec?вҖқ вҶ’ Often lower-tailed on fraction defective
  • О”вӮҖ = 0 unless problem says вҖңtest if ОјвӮҒ вҲ’ ОјвӮӮ = some valueвҖқ

## "Which t?" вҖ” The Hardest Decision

| Clue in Problem | Use |
|-----------------|-----|
| "before and after on the same unit" | Paired t |
| "same subjects tested twice" | Paired t |
| "two independent groups / samples" | 2-sample t (pooled or Welch) |
| "assume equal variances" | Pooled t |
| "do not assume equal variances" | Welch t |
| "assume Пғ is known" | Z (not t) |
| large samples (nвӮҒ, nвӮӮ вүҘ 40) | Z acceptable |
