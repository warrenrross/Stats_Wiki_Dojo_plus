1.1 standard-error

The universal SE formula (SD of a sampling distribution) appears in every CI, test statistic, regression coefficient, and control chart limit. Every procedure page that computes a test statistic or interval should link here.

Pages that should link to standard-error:

Source page Reason
confidence-intervals CI formula divides by SE
hypothesis-testing-overview Test stat = (estimate − null) / SE
two-sample-tests SE of difference of means
regression-slr SE of β̂₁ and β̂₀
regression-mlr SE of βⱼ from MLR
regression-ht t = β̂ⱼ / SE(β̂ⱼ)
regression-ci CI on βⱼ uses SE; CI/PI bands use SE of mean response
control-charts σ̂ = R̄/d₂ or s̄/c₄ plays the SE role in control limits
ht-two-sample-z-means SE of X̄₁ − X̄₂ with known variances
ht-two-sample-t-means Pooled SE and Welch SE
ht-paired-t SE of D̄ = s_D / √n
power-sample-size n determines SE; n chosen to achieve target power

1.2 ss-decomposition

SS_T = SS_Model + SS_Error is the identity underlying every ANOVA table. Every page that builds or interprets an ANOVA table should link here.

Pages that should link to ss-decomposition:

Source page Reason
regression-slr SS_T = SS_R + SS_E is the SLR decomposition
regression-mlr Same identity; R² = SS_R / SS_T
regression-ht F-test = MS_R / MS_E; both come from this partition
crd-one-way-anova ANOVA identity: SS_T = SS_Trt + SS_E
rcbd-blocking Three-way partition: SS_T = SS_Trt + SS_Blk + SS_E
factorial-anova SS_Model split into main effects and interactions
2k-factorial-design Each effect gets its own SS row
anova-table-examples Every ANOVA table completion problem uses this identity

1.3 degrees-of-freedom

The df formula changes with every test and design. Every HT, CI, and ANOVA page has a specific df formula that belongs here.

Pages that should link to degrees-of-freedom:

Source page Reason
confidence-intervals t critical value uses df = n − 1
hypothesis-testing-overview df for one-sample t and chi-square
two-sample-tests Pooled df = n₁+n₂−2; Welch df from Satterthwaite
chi-square GoF df = k−1−p; independence df = (r−1)(c−1)
regression-slr df_Reg = 1, df_Res = n−2
regression-mlr df_Reg = k, df_Res = n−k−1
regression-ht t-test df and F-test df
crd-one-way-anova df_error = N − a
rcbd-blocking df_error = (a−1)(b−1)
factorial-anova df for main effects, interactions, error
2k-factorial-design df = 1 per effect; df_error from replicates
ht-one-sample-t-mean df = n − 1
ht-one-sample-chisq-variance df = n − 1
ht-goodness-of-fit df = k − 1 − p
ht-independence df = (r−1)(c−1)

Pages that should link to variance-estimation:

Source page Reason
confidence-intervals χ² interval on σ²
two-sample-tests Pooled s² formula
regression-slr MSE = σ̂² in SLR
regression-mlr MSE in MLR
regression-ht MSE feeds F and t tests
regression-ci MSE inside CI and PI formulas
control-charts σ̂ from R̄/d₂ or s̄/c₄
process-capability σ̂ feeds Cp and Cpk
crd-one-way-anova MS_Error = σ̂²
rcbd-blocking MS_Error = σ̂²
random-effects-model Variance components σ̂²τ = (MS_Trt − MS_E) / n
